刚体动力学：从基矢旋转到角速度推导

[!abstract] 符号约定的由来

  • 基矢的旋转（主动）：假设初始基矢为 e(0)\mathbf{e}(0)e(0)，随时间旋转后的基矢为 e(t)\mathbf{e}(t)e(t)。若旋转算符为 UUU，则有 e(t)=Ue(0)\mathbf{e}(t) = U \mathbf{e}(0)e(t)=Ue(0)。

  • 分量的变换（被动：对于一个空间中固定的物理向量 V\mathbf{V}V，它在不同基矢下的分量满足不变性：V=xTe(0)=x′Te(t)\mathbf{V} = \mathbf{x}^T \mathbf{e}(0) = \mathbf{x}'^T \mathbf{e}(t)V=xTe(0)=x′Te(t)。

  • 在欧拉角的定义中，坐标系eˉ\bar{e}eˉ变成了e′e^{'}e′,然后e′′e^{''}e′′,旋转矩阵应该是

    Rz′′(ψ)Rx′(θ)Rz(ϕ)R_{z^{''}}(\psi)R_{x^{'}}(\theta)R_{z^{}}(\phi)Rz′′​(ψ)Rx′​(θ)Rz​(ϕ)

    这些轴并不在固定的空间轴上，会很麻烦，但是我们可以证明这个旋转矩阵等价于

    Rz(ϕ)Rx(θ)Rz(ψ)R_{z}(\phi)R_{x}(\theta)R_{z}(\psi)Rz​(ϕ)Rx​(θ)Rz​(ψ)

    详情可以参考樱井纯的现代量子力学第三章

  • 以上矩阵是改变基矢的，我们要把它变成改变矢量在基矢下的分量的。很简单，根据

    xTe=x′  TRe\mathbf{x}^T \mathbf{e} = \mathbf{x}'^{\;T} R \mathbf{e}xTe=x′TRe

    可以得到

    x′=R−1x\mathbf{x}'=R^{-1}\mathbf{x}x′=R−1x

  • 算符形式：
    旋转矩阵 UUU 描述的是分量的变换，为了对应 Z-X-Z 欧拉角序列，取：

    R=e−ψJ3e−θJ1e−ϕJ3R = e^{-\psi J_3} e^{-\theta J_1} e^{-\phi J_3} R=e−ψJ3​e−θJ1​e−ϕJ3​

    相应地，角速度张量定义为 Ω=−R˙R−1\Omega = -\dot{R} R^{-1}Ω=−R˙R−1。详细推导见高显经典力学

刚体动力学：重对称陀螺的拉格朗日量

[!abstract] 物理模型

  • 对称性：假设刚体绕其自转轴（随体系 zzz 轴）对称，即转动惯量满足 I1=I2≠I3I_1 = I_2 \neq I_3I1​=I2​=I3​。
  • 坐标系：使用 Z-X-Z 欧拉角 (ϕ,θ,ψ)(\phi, \theta, \psi)(ϕ,θ,ψ)。
  • 势能：考虑在重力场中，质量中心距离固定点距离为 lll，势能 V=Mglcos⁡θV = Mgl \cos \thetaV=Mglcosθ。

4. 动能项的构造 TTT

根据之前的推导，随体系下的角速度分量为：

  • ωx=θ˙cos⁡ψ+ϕ˙sin⁡θsin⁡ψ\omega_x = \dot{\theta} \cos \psi + \dot{\phi} \sin \theta \sin \psiωx​=θ˙cosψ+ϕ˙​sinθsinψ
  • ωy=−θ˙sin⁡ψ+ϕ˙sin⁡θcos⁡ψ\omega_y = -\dot{\theta} \sin \psi + \dot{\phi} \sin \theta \cos \psiωy​=−θ˙sinψ+ϕ˙​sinθcosψ
  • ωz=ψ˙+ϕ˙cos⁡θ\omega_z = \dot{\psi} + \dot{\phi} \cos \thetaωz​=ψ˙​+ϕ˙​cosθ

转动动能公式为 T=12(I1ωx2+I2ωy2+I3ωz2)T = \frac{1}{2} (I_1 \omega_x^2 + I_2 \omega_y^2 + I_3 \omega_z^2)T=21​(I1​ωx2​+I2​ωy2​+I3​ωz2​)。由于 I1=I2I_1 = I_2I1​=I2​，我们可以简化前两项：

4.1 简化 ωx2+ωy2\omega_x^2 + \omega_y^2ωx2​+ωy2​

ωx2+ωy2=(θ˙cos⁡ψ+ϕ˙sin⁡θsin⁡ψ)2+(−θ˙sin⁡ψ+ϕ˙sin⁡θcos⁡ψ)2=θ˙2(cos⁡2ψ+sin⁡2ψ)+ϕ˙2sin⁡2θ(sin⁡2ψ+cos⁡2ψ)+2θ˙ϕ˙sin⁡θ(cos⁡ψsin⁡ψ−sin⁡ψcos⁡ψ)=θ˙2+ϕ˙2sin⁡2θ\begin{aligned} \omega_x^2 + \omega_y^2 &= (\dot{\theta} \cos \psi + \dot{\phi} \sin \theta \sin \psi)^2 + (-\dot{\theta} \sin \psi + \dot{\phi} \sin \theta \cos \psi)^2 \\ &= \dot{\theta}^2 (\cos^2 \psi + \sin^2 \psi) + \dot{\phi}^2 \sin^2 \theta (\sin^2 \psi + \cos^2 \psi) + 2\dot{\theta}\dot{\phi} \sin \theta (\cos \psi \sin \psi - \sin \psi \cos \psi) \\ &= \dot{\theta}^2 + \dot{\phi}^2 \sin^2 \theta \end{aligned} ωx2​+ωy2​​=(θ˙cosψ+ϕ˙​sinθsinψ)2+(−θ˙sinψ+ϕ˙​sinθcosψ)2=θ˙2(cos2ψ+sin2ψ)+ϕ˙​2sin2θ(sin2ψ+cos2ψ)+2θ˙ϕ˙​sinθ(cosψsinψ−sinψcosψ)=θ˙2+ϕ˙​2sin2θ​

[!note] 关键发现

  • 交叉项抵消了，且 ψ\psiψ 从前两项中消失了。这说明对于对称陀螺，横向动能与自转角度无关。

5. 守恒量分析 (Conserved Quantities)

观察拉格朗日量 L(θ,θ˙,ϕ˙,ψ˙)L(\theta, \dot{\theta}, \dot{\phi}, \dot{\psi})L(θ,θ˙,ϕ˙​,ψ˙​)，可以发现两个循环坐标（Cyclic Coordinates）：

  1. 进动角 ϕ\phiϕ 守恒：∂L∂ϕ=0\frac{\partial L}{\partial \phi} = 0∂ϕ∂L​=0，对应空间 zzz 轴角动量守恒：

    pϕ=∂L∂ϕ˙=(I1sin⁡2θ+I3cos⁡2θ)ϕ˙+I3ψ˙cos⁡θ=constp_\phi = \frac{\partial L}{\partial \dot{\phi}} = (I_1 \sin^2 \theta + I_3 \cos^2 \theta) \dot{\phi} + I_3 \dot{\psi} \cos \theta = \text{const} pϕ​=∂ϕ˙​∂L​=(I1​sin2θ+I3​cos2θ)ϕ˙​+I3​ψ˙​cosθ=const

  2. 自转角 ψ\psiψ 守恒：∂L∂ψ=0\frac{\partial L}{\partial \psi} = 0∂ψ∂L​=0，对应随体 zzz 轴角动量守恒：

    pψ=∂L∂ψ˙=I3(ψ˙+ϕ˙cos⁡θ)=I3ωz=constp_\psi = \frac{\partial L}{\partial \dot{\psi}} = I_3 (\dot{\psi} + \dot{\phi} \cos \theta) = I_3 \omega_z = \text{const} pψ​=∂ψ˙​∂L​=I3​(ψ˙​+ϕ˙​cosθ)=I3​ωz​=const

  3. 能量守恒：由于 LLL 不显含时间 ttt，总机械能 E=T+VE = T + VE=T+V 守恒。
