1. 基础篇：从力学到几何 (Time-Independent)

将哈密顿力学用几何语言重写，核心在于建立“物理量”与“几何对象”的映射。

1.1 核心概念对照表

经典分析力学 (Mechanics) 辛几何 (Symplectic Geometry) 数学表达
相空间 (q,p)(q, p)(q,p) 辛流形 (Symplectic Manifold) MMM
物理结构 辛形式 (Symplectic Form) ω=dq∧dp\omega = dq \wedge dpω=dq∧dp
哈密顿量 HHH 标量场 (Hamiltonian Function) H:M→RH: M \to \mathbb{R}H:M→R
运动状态 哈密顿向量场 (Vector Field) XHX_HXH​
运动方程 卡当方程 (Cartan’s Eq) iXHω=dHi_{X_H} \omega = dHiXH​​ω=dH

[!abstract] 基本运动方程

iXHω=dHi_{X_H} \omega = dH iXH​​ω=dH

  • dHdHdH：告诉我们能量怎么变。
  • ω\omegaω：定义了空间的曲率/结构。
  • iXHi_{X_H}iXH​​：通过内部乘积将它们联系起来，生成运动轨迹。

1.2 实例：一维谐振子

[!example] 问题设定
哈密顿量 H=12p2+12q2H = \frac{1}{2}p^2 + \frac{1}{2}q^2H=21​p2+21​q2。求运动方程。

  1. 计算 dHdHdH：dH=qdq+pdpdH = q dq + p dpdH=qdq+pdp
  2. 计算 iXHωi_{X_H} \omegaiXH​​ω：设 XH=A∂q+B∂pX_H = A\partial_q + B\partial_pXH​=A∂q​+B∂p​，则 iXH(dq∧dp)=Adp−Bdqi_{X_H}(dq \wedge dp) = A dp - B dqiXH​​(dq∧dp)=Adp−Bdq
  3. 几何方程求解：

    Adp−Bdq=qdq+pdpA dp - B dq = q dq + p dp Adp−Bdq=qdq+pdp

    对比系数得：A=pA = pA=p (即 q˙=p\dot{q}=pq˙​=p)，−B=q-B=q−B=q (即 p˙=−q\dot{p}=-qp˙​=−q)。

2. 进阶篇：含时系统的几何化 (Time-Dependent)

当哈密顿量显含时间 H(q,p,t)H(q, p, t)H(q,p,t) 时，时间不再是参数，而是坐标。我们需要升级到扩展相空间。

2.1 扩展几何架构

  • 流形：M~=M×R\tilde{M} = M \times \mathbb{R}M~=M×R (坐标 q,p,tq, p, tq,p,t)。
  • 庞加莱-卡当形式 (Poincaré-Cartan form)：

    Θ=pdq−H(q,p,t)dt\Theta = p dq - H(q, p, t) dt Θ=pdq−H(q,p,t)dt

  • 闭形式 (2-form)：

    Ω=dΘ=dp∧dq−dH∧dt\Omega = d\Theta = dp \wedge dq - dH \wedge dt Ω=dΘ=dp∧dq−dH∧dt

  • 动力学条件：寻找向量场 X~\tilde{X}X~ 使得：

    iX~Ω=0i_{\tilde{X}} \Omega = 0 iX~​Ω=0

2.2 实例：参数激励谐振子 (变频振子)

[!example] 问题设定
频率随时间变化：H=p22m+12mω2(t)q2H = \frac{p^2}{2m} + \frac{1}{2} m \omega^2(t) q^2H=2mp2​+21​mω2(t)q2。

1. 计算几何结构 Ω\OmegaΩ

Ω=dp∧dq−(mω2q dq+pm dp)∧dt\Omega = dp \wedge dq - \left( m \omega^2 q \, dq + \frac{p}{m} \, dp \right) \wedge dt Ω=dp∧dq−(mω2qdq+mp​dp)∧dt

2. 求解 iX~Ω=0i_{\tilde{X}} \Omega = 0iX~​Ω=0
设 X~=q˙∂q+p˙∂p+∂t\tilde{X} = \dot{q}\partial_q + \dot{p}\partial_p + \partial_tX~=q˙​∂q​+p˙​∂p​+∂t​，代入计算：

  • dqdqdq 的系数方程：p˙+mω2q=0  ⟹  p˙=−mω2(t)q\dot{p} + m\omega^2 q = 0 \implies \dot{p} = -m\omega^2(t)qp˙​+mω2q=0⟹p˙​=−mω2(t)q
  • dpdpdp 的系数方程：−q˙+pm=0  ⟹  q˙=pm-\dot{q} + \frac{p}{m} = 0 \implies \dot{q} = \frac{p}{m}−q˙​+mp​=0⟹q˙​=mp​

结果：完美导出了含时系统的正则方程。

3. 几何视角的威力 (Why do this?)

为什么我们要用这么复杂的几何语言？

A. 刘维尔定理 (Liouville’s Theorem)

相体积守恒变成了纯几何推论。流体沿着哈密顿向量场 XHX_HXH​ 流动时，辛体积形式 ω\omegaω 不变：

LXHω=iXHdω⏟0+d(iXHω⏟dH)=d2H=0\mathcal{L}_{X_H} \omega = i_{X_H} \underbrace{d\omega}_{0} + d(\underbrace{i_{X_H} \omega}_{dH}) = d^2H = 0 LXH​​ω=iXH​​0dω​​+d(dHiXH​​ω​​)=d2H=0

[!tip] 结论
不需要计算雅可比行列式，由外微分的幂零性 (d2=0d^2=0d2=0) 直接保证了统计力学的基石。

B. 泊松括号与对称性

泊松括号是辛形式的逆运算：{f,g}=ω(Xf,Xg)\{f, g\} = \omega(X_f, X_g){f,g}=ω(Xf​,Xg​)。
这让寻找守恒量（Noether 定理）变成了寻找流形上的几何对称性。
