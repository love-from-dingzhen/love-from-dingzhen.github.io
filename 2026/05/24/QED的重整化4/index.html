光子传播子 Counterterm 的推导

光子自能与电子自能类似。

光子自由传播时，先变成一个虚的电子-正电子对，随后这个电子-正电子对又重新湮灭成光子。

(a) 光子自由传播

(b) 光子自能图

光子的自由传播子这里取为横向形式：

iGtreeμν(p)=−ip2+iϵ(gμν−pμpνp2).iG_{\rm tree}^{\mu\nu}(p) = \frac{-i}{p^2+i\epsilon} \left( g^{\mu\nu} - \frac{p^\mu p^\nu}{p^2} \right). iGtreeμν​(p)=p2+iϵ−i​(gμν−p2pμpν​).

为了方便，定义横向投影算符

PTμν(p)=gμν−pμpνp2.P_T^{\mu\nu}(p) = g^{\mu\nu} - \frac{p^\mu p^\nu}{p^2}. PTμν​(p)=gμν−p2pμpν​.

那么自由光子传播子可以简写为

iGtreeμν(p)=−ip2+iϵPTμν(p).iG_{\rm tree}^{\mu\nu}(p) = \frac{-i}{p^2+i\epsilon} P_T^{\mu\nu}(p). iGtreeμν​(p)=p2+iϵ−i​PTμν​(p).

光子自能修正对应的是：

iGselfμν(p)=iGtreeμα(p) iΠαβ(2)(p) iGtreeβν(p).iG_{\rm self}^{\mu\nu}(p) = iG_{\rm tree}^{\mu\alpha}(p) \, i\Pi_{\alpha\beta}^{(2)}(p) \, iG_{\rm tree}^{\beta\nu}(p). iGselfμν​(p)=iGtreeμα​(p)iΠαβ(2)​(p)iGtreeβν​(p).

也就是说，光子自能图不是单独一个传播子，而是：

光子传播子×真空极化插入×光子传播子.\text{光子传播子} \times \text{真空极化插入} \times \text{光子传播子}. 光子传播子×真空极化插入×光子传播子.

光子的 1-loop 自能图也叫真空极化图，它的形式为

iΠ2μν(p)=(−1)(−ie)2∫d4k(2π)4Tr[γμi(k̸+m)k2−m2+iϵγνi(k̸+p̸+m)(k+p)2−m2+iϵ].i\Pi_2^{\mu\nu}(p) = (-1)(-ie)^2 \int \frac{d^4 k}{(2\pi)^4} \text{Tr} \left[ \gamma^\mu \frac{i(\not k+m)}{k^2-m^2+i\epsilon} \gamma^\nu \frac{i(\not k+\not p+m)} {(k+p)^2-m^2+i\epsilon} \right]. iΠ2μν​(p)=(−1)(−ie)2∫(2π)4d4k​Tr[γμk2−m2+iϵi(k+m)​γν(k+p)2−m2+iϵi(k+p+m)​].

这里前面的 (−1)(-1)(−1) 来自闭合费米子圈。

由于真空极化图是发散的，所以我们需要引入光子场的重整化因子 Z3Z_3Z3​。

1. Bare field 与 renormalized field

光子场的 bare field 和 renormalized field 的关系为

ARμ(x)=1Z3A0μ(x).A_R^\mu(x) = \frac{1}{\sqrt{Z_3}}A_0^\mu(x). ARμ​(x)=Z3​​1​A0μ​(x).

等价地，

A0μ(x)=Z3ARμ(x).A_0^\mu(x) = \sqrt{Z_3}A_R^\mu(x). A0μ​(x)=Z3​​ARμ​(x).

2. 定义 bare Green 函数与 renormalized Green 函数

bare 光子场的二点函数定义为

⟨A0μ(x)A0ν(y)⟩=i∫d4p(2π)4e−ip(x−y)Gbareμν(p).\langle A_0^\mu(x)A_0^\nu(y)\rangle = i\int \frac{d^4p}{(2\pi)^4} e^{-ip(x-y)} G_{\rm bare}^{\mu\nu}(p). ⟨A0μ​(x)A0ν​(y)⟩=i∫(2π)4d4p​e−ip(x−y)Gbareμν​(p).

renormalized 光子场的二点函数定义为

⟨ARμ(x)ARν(y)⟩=i∫d4p(2π)4e−ip(x−y)GRμν(p).\langle A_R^\mu(x)A_R^\nu(y)\rangle = i\int \frac{d^4p}{(2\pi)^4} e^{-ip(x-y)} G_R^{\mu\nu}(p). ⟨ARμ​(x)ARν​(y)⟩=i∫(2π)4d4p​e−ip(x−y)GRμν​(p).

由于

ARμ=1Z3A0μ,ARν=1Z3A0ν,A_R^\mu = \frac{1}{\sqrt{Z_3}}A_0^\mu, \qquad A_R^\nu = \frac{1}{\sqrt{Z_3}}A_0^\nu, ARμ​=Z3​​1​A0μ​,ARν​=Z3​​1​A0ν​,

所以

⟨ARμ(x)ARν(y)⟩=1Z3⟨A0μ(x)A0ν(y)⟩.\langle A_R^\mu(x)A_R^\nu(y)\rangle = \frac{1}{Z_3} \langle A_0^\mu(x)A_0^\nu(y)\rangle. ⟨ARμ​(x)ARν​(y)⟩=Z3​1​⟨A0μ​(x)A0ν​(y)⟩.

因此

GRμν(p)=1Z3Gbareμν(p).G_R^{\mu\nu}(p) = \frac{1}{Z_3} G_{\rm bare}^{\mu\nu}(p). GRμν​(p)=Z3​1​Gbareμν​(p).

3. 用 bare propagator 写出 renormalized propagator

现在把 renormalized photon propagator 写成 bare propagator 的形式：

iGRμν(p)=1Z3iGtreeμν(p)+loops.iG_R^{\mu\nu}(p) = \frac{1}{Z_3} iG_{\rm tree}^{\mu\nu}(p) + \text{loops}. iGRμν​(p)=Z3​1​iGtreeμν​(p)+loops.

代入

Z3=1+δ3,Z_3=1+\delta_3, Z3​=1+δ3​,

得到

iGRμν(p)=11+δ3iGtreeμν(p)+loops.iG_R^{\mu\nu}(p) = \frac{1}{1+\delta_3} iG_{\rm tree}^{\mu\nu}(p) + \text{loops}. iGRμν​(p)=1+δ3​1​iGtreeμν​(p)+loops.

接下来对 δ3\delta_3δ3​ 做一阶展开。

iGRμν(p)=iGtreeμν(p)−δ3iGtreeμν(p)+loops+O(e4).iG_R^{\mu\nu}(p) = iG_{\rm tree}^{\mu\nu}(p) - \delta_3 iG_{\rm tree}^{\mu\nu}(p) + \text{loops} + \mathcal O(e^4). iGRμν​(p)=iGtreeμν​(p)−δ3​iGtreeμν​(p)+loops+O(e4).

现在关键是把第二项

−δ3iGtreeμν(p)-\delta_3 iG_{\rm tree}^{\mu\nu}(p) −δ3​iGtreeμν​(p)

写成

光子传播子×counterterm 插入×光子传播子.\text{光子传播子} \times \text{counterterm 插入} \times \text{光子传播子}. 光子传播子×counterterm 插入×光子传播子.

也就是说，我们希望找到一个插入项 iCαβ(p)iC^{\alpha\beta}(p)iCαβ(p)，使得

iGtreeμα(p) iCαβ(p) iGtreeβν(p)=−δ3iGtreeμν(p).iG_{\rm tree}^{\mu\alpha}(p) \, iC^{\alpha\beta}(p) \, iG_{\rm tree}^{\beta\nu}(p) = -\delta_3 iG_{\rm tree}^{\mu\nu}(p). iGtreeμα​(p)iCαβ(p)iGtreeβν​(p)=−δ3​iGtreeμν​(p).

因为

iGtreeμν(p)=−ip2+iϵPTμν(p),iG_{\rm tree}^{\mu\nu}(p) = \frac{-i}{p^2+i\epsilon} P_T^{\mu\nu}(p), iGtreeμν​(p)=p2+iϵ−i​PTμν​(p),

而横向投影算符满足

PTμαPTαν=PTμν.P_T^{\mu\alpha}P_{T}^{\alpha \nu} = P_T^{\mu\nu}. PTμα​PTαν​=PTμν​.

所以如果取

iCαβ(p)=−iδ3p2PTαβ(p),iC^{\alpha\beta}(p) = -i\delta_3 p^2 P_{T}^{\alpha\beta}(p), iCαβ(p)=−iδ3​p2PTαβ​(p),

那么

iGtreeμα(p) iCαβ(p) iGtreeβν(p)=−ip2PTμα[−iδ3p2PTαβ]−ip2PTβν.iG_{\rm tree}^{\mu\alpha}(p) \, iC^{\alpha\beta}(p) \, iG_{\rm tree}^{\beta\nu}(p) = \frac{-i}{p^2} P_T^{\mu\alpha} \left[ -i\delta_3 p^2P_{T}^{\alpha\beta} \right] \frac{-i}{p^2} P_T^{\beta\nu}. iGtreeμα​(p)iCαβ(p)iGtreeβν​(p)=p2−i​PTμα​[−iδ3​p2PTαβ​]p2−i​PTβν​.

先看前面的系数：

−ip2(−iδ3p2)−ip2=iδ3p2.\frac{-i}{p^2} (-i\delta_3 p^2) \frac{-i}{p^2} = \frac{i\delta_3}{p^2}. p2−i​(−iδ3​p2)p2−i​=p2iδ3​​.

再看投影算符部分：

PTμαPTαβPTβν=PTμν.P_T^{\mu\alpha} P_T^{\alpha\beta} P_T^{\beta\nu} = P_T^{\mu\nu}. PTμα​PTαβ​PTβν​=PTμν​.

因此

iGtreeμα(p) iCαβ(p) iGtreeβν(p)=iδ3p2PTμν.iG_{\rm tree}^{\mu\alpha}(p) \, iC^{\alpha\beta}(p) \, iG_{\rm tree}^{\beta\nu}(p) = \frac{i\delta_3}{p^2} P_T^{\mu\nu}. iGtreeμα​(p)iCαβ(p)iGtreeβν​(p)=p2iδ3​​PTμν​.

另一方面，

−δ3iGtreeμν(p)=−δ3(−ip2PTμν)=iδ3p2PTμν.-\delta_3 iG_{\rm tree}^{\mu\nu}(p) = -\delta_3 \left( \frac{-i}{p^2} P_T^{\mu\nu} \right) = \frac{i\delta_3}{p^2} P_T^{\mu\nu}. −δ3​iGtreeμν​(p)=−δ3​(p2−i​PTμν​)=p2iδ3​​PTμν​.

两边正好相等。

所以光子传播子上的 counterterm 插入就是

iCαβ(p)=−iδ3p2PTαβ(p).iC^{\alpha\beta}(p) = -i\delta_3 p^2P_{T}^{\alpha\beta}(p). iCαβ(p)=−iδ3​p2PTαβ​(p).

再把 PTP_TPT​ 展开：

p2PTμν(p)=p2(gμν−pμpνp2)=p2gμν−pμpν.p^2P_T^{\mu\nu}(p) = p^2 \left( g^{\mu\nu} - \frac{p^\mu p^\nu}{p^2} \right) = p^2g^{\mu\nu} - p^\mu p^\nu. p2PTμν​(p)=p2(gμν−p2pμpν​)=p2gμν−pμpν.

因此

iCμν(p)=−iδ3(p2gμν−pμpν).iC^{\mu\nu}(p) = -i\delta_3 \left( p^2g^{\mu\nu} - p^\mu p^\nu \right). iCμν(p)=−iδ3​(p2gμν−pμpν).

这一项

−iδ3(p2gμν−pμpν)-i\delta_3 \left( p^2g^{\mu\nu} - p^\mu p^\nu \right) −iδ3​(p2gμν−pμpν)

就是光子传播子上的 counterterm 插入。

4. 更方便地把整体展开写成

于是 renormalized photon propagator 可以写成

iGRμν(p)=iGtreeμν(p)+iGtreeμα(p)[−iδ3(p2gαβ−pαpβ)]iGtreeβν(p)+loops+O(e4).iG_R^{\mu\nu}(p) = iG_{\rm tree}^{\mu\nu}(p) + iG_{\rm tree}^{\mu\alpha}(p) \left[ -i\delta_3 \left( p^2g^{\alpha\beta} - p^\alpha p^\beta \right) \right] iG_{\rm tree}^{\beta\nu}(p) + \text{loops} + \mathcal O(e^4). iGRμν​(p)=iGtreeμν​(p)+iGtreeμα​(p)[−iδ3​(p2gαβ−pαpβ)]iGtreeβν​(p)+loops+O(e4).

这一项

−iδ3(p2gαβ−pαpβ)-i\delta_3 \left( p^2g^{\alpha\beta} - p^\alpha p^\beta \right) −iδ3​(p2gαβ−pαpβ)

就是由 Z3=1+δ3Z_3=1+\delta_3Z3​=1+δ3​ 展开得到的光子传播子 counterterm 插入。

5. 加上 1-loop 光子自能图

光子的 1-loop 自能修正写作：

iΠ2μν(p)=−i(p2gμν−pμpν)Π2(p2).i\Pi_2^{\mu\nu}(p) =-i\left( p^2g^{\mu\nu} - p^\mu p^\nu \right) \Pi_2(p^2). iΠ2μν​(p)=−i(p2gμν−pμpν)Π2​(p2).

原因会写

因此完整的 e2e^2e2 阶 renormalized photon propagator 为：

iGRμν(p)=iGtreeμν(p)+iGtreeμα(p)[iΠ2αβ(p)−iδ3(p2gαβ−pαpβ)]iGtreeβν(p)+O(e4).iG_R^{\mu\nu}(p) = iG_{\rm tree}^{\mu\nu}(p) + iG_{\rm tree}^{\mu\alpha}(p) \left[ i\Pi_{2}^{\alpha\beta}(p) - i\delta_3 \left( p^2g^{\alpha\beta} - p^\alpha p^\beta \right) \right] iG_{\rm tree}^{\beta\nu}(p) + \mathcal O(e^4). iGRμν​(p)=iGtreeμν​(p)+iGtreeμα​(p)[iΠ2αβ​(p)−iδ3​(p2gαβ−pαpβ)]iGtreeβν​(p)+O(e4).

iΠ2αβ(p)−iδ3(p2gαβ−pαpβ)=−i(p2gαβ−pαpβ)[Π2(p2)+δ3].i\Pi_{2}^{\alpha\beta}(p) - i\delta_3 \left( p^2g^{\alpha\beta} - p^\alpha p^\beta \right) = -i \left( p^2g^{\alpha\beta} - p^\alpha p^\beta \right) \left[ \Pi_2(p^2)+\delta_3 \right]. iΠ2αβ​(p)−iδ3​(p2gαβ−pαpβ)=−i(p2gαβ−pαpβ)[Π2​(p2)+δ3​].

因此为了抵消真空极化中的发散部分，应当取

δ3=−Π2(p2)∣div.\delta_3 = -\left.\Pi_2(p^2)\right|_{\rm div}. δ3​=−Π2​(p2)​div​.

这样

Π2(p2)+δ3\Pi_2(p^2)+\delta_3 Π2​(p2)+δ3​

就是有限的。

也就是说，光子传播子的 counterterm 正是用来抵消 vacuum polarization 的发散部分。

6. 最终结果

光子传播子的 counterterm 插入为

−iδ3(p2gμν−pμpν).-i\delta_3 \left( p^2g^{\mu\nu} - p^\mu p^\nu \right). −iδ3​(p2gμν−pμpν).

完整的 e2e^2e2 阶光子传播子为

iGRμν(p)=iGtreeμν(p)+iGtreeμα(p)[iΠ2αβ(p)−iδ3(p2gαβ−pαpβ)]iGtreeβν(p)+O(e4).iG_R^{\mu\nu}(p) = iG_{\rm tree}^{\mu\nu}(p) + iG_{\rm tree}^{\mu\alpha}(p) \left[ i\Pi_{2}^{\alpha\beta}(p) - i\delta_3 \left( p^2g^{\alpha\beta} - p^\alpha p^\beta \right) \right] iG_{\rm tree}^{\beta\nu}(p) + \mathcal O(e^4). iGRμν​(p)=iGtreeμν​(p)+iGtreeμα​(p)[iΠ2αβ​(p)−iδ3​(p2gαβ−pαpβ)]iGtreeβν​(p)+O(e4).

这和电子传播子的结构完全类似：

自由传播子+自由传播子×(loop 自能+counterterm)×自由传播子.\text{自由传播子} + \text{自由传播子} \times \left( \text{loop 自能} + \text{counterterm} \right) \times \text{自由传播子}. 自由传播子+自由传播子×(loop 自能+counterterm)×自由传播子.
