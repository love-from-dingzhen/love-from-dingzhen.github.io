1️⃣ Wireless Communication Overview

Power and Decibel System

dBW=10lg⁡(P/1W)\text{dBW} = 10\lg(P/1\text{W}) dBW=10lg(P/1W)

dBm=10lg⁡(P/1mW)\text{dBm} = 10\lg(P/1\text{mW}) dBm=10lg(P/1mW)

  • 1 W = 0 dBW = 30 dBm

Gain/Attenuation

Gain(dB)=10lg⁡PoutPin=Pout(dBm)−Pin(dBm)\text{Gain(dB)} = 10\lg\frac{P_{out}}{P_{in}} = P_{out}(\text{dBm}) - P_{in}(\text{dBm}) Gain(dB)=10lgPin​Pout​​=Pout​(dBm)−Pin​(dBm)

Operation Rules

  • dBm ± dB = dBm
  • dBm − dBm = dB
  • Example: 2 dBm + 5 dB = 7 dBm

EIRP & ERP

  • EIRP = ERP + 2.15 dB
  • dBi = dBd + 2.15 dB
  • EIRP = Pt + Gt (dBm)

Shannon Capacity Theory

C=Blog⁡2(1+SN)[bps]C = B\log_2\left(1 + \frac{S}{N}\right) \quad [\text{bps}] C=Blog2​(1+NS​)[bps]

Spectral Efficiency

ηB=RB[bit/s/Hz]\eta_B = \frac{R}{B} \quad [\text{bit/s/Hz}] ηB​=BR​[bit/s/Hz]

Typical Calculation

  • B=22MHz, SNR=10dB→S/N=10
  • C = 22·log₂(1+10) ≈ 76 Mbps

Signal-to-Noise Ratio System

  • SNR = S/N (Signal/Noise)
  • SIR = C/I (Signal/Interference)
  • SINR = S/(N+I)

Application Scenarios

  • SNR → AWGN Channel
  • SIR → Cellular Co-channel Interference
  • SINR → Practical Comprehensive Evaluation

Mobile Communication Evolution

  • 1G→5G Multiple Access Evolution

1G

  • FDMA / Analog / AMPS

2G

  • TDMA / GSM / 14.4K~171Kbps

3G

  • CDMA / WCDMA / 14.4Mbps

4G

  • OFDMA / LTE-A / 100~300Mbps

5G

  • New OFDM / eMBB+URLLC+mMTC / 20Gbps

2️⃣ Cellular Systems

Frequency Reuse

N=i2+ij+j2N = i^2 + ij + j^2 N=i2+ij+j2

Q=DR=3NQ = \frac{D}{R} = \sqrt{3N} Q=RD​=3N​

Common N Values

  • N=3(i=1,j=1), Q=3.00
  • N=7(i=2,j=1), Q=4.58
  • N=12(i=2,j=2), Q=6.00
  • N=19(i=3,j=2), Q=7.55

System Capacity

  • C = M × k × N = M × S
  • k = S/N (Channels per cell)
  • M: Reuse times

Co-channel Interference

SI=(3N)ni0\frac{S}{I} = \frac{(\sqrt{3N})^n}{i_0} IS​=i0​(3N​)n​

  • i₀=6(First-tier interference cells), n: Path loss exponent

Handover Mechanisms

Δ=Pr,ho−Pr,min\Delta = P_{r,ho} - P_{r,min} Δ=Pr,ho​−Pr,min​

Hard Handover

  • Break-before-make
  • GSM / Must change frequency

Soft Handover

  • Make-before-break
  • CDMA / Same frequency maintained / MSC selects best

MAHO

  • Mobile Assisted Handover
  • MS provides measurement reports to BS

Trunking and Erlang B

  • A = λ·H [Erlang]
  • Aᵤ = λ·H (Per user)
  • U = A/Aᵤ

Erlang B Formula

Pb=AC/C!∑k=0CAk/k!P_b = \frac{A^C/C!}{\sum_{k=0}^{C} A^k/k!} Pb​=∑k=0C​Ak/k!AC/C!​

  • Table lookup: Given C and GOS, find A

Typical Table Values

  • C=10,GOS=1%→A=4.46
  • C=20,GOS=1%→A=12.0
  • C=40,GOS=1%→A=29.0

Example Problem

  • C=19,GOS=2%,A=12Erl
  • λ=2calls/h,H=3min=0.05h
  • Aᵤ=0.1, U=120 users/cell

Coverage Improvement

  • Cell Splitting + Sectorization

Cell Splitting

  • Radius halved→Boundary PL reduced by 12dB(n=4)
  • Capacity increased 4 times
  • Cost: New base stations

Sectorization

  • 120° sector: 57ch→19ch/sector
  • 11.2Erl/sector→336calls/h
  • 3sector→1008calls/h(cell)

Trade-offs

  • Splitting: Capacity↑ Cost↑
  • Sectorization: SIR↑ Trunking efficiency↓

3️⃣ Large-Scale Fading

Friis Free Space

Pr=PtGtGrλ2(4πd)2LP_r = \frac{P_t G_t G_r \lambda^2}{(4\pi d)^2 L} Pr​=(4πd)2LPt​Gt​Gr​λ2​

Path Loss (dB)

  • PL = 32.45 + 20lg f(MHz) + 20lg d(km)

Key Patterns

  • f×2 or d×2 → PL+6dB
  • d×10 → PL+20dB(20dB/decade)

Far-field Condition

df=2D2λd_f = \frac{2D^2}{\lambda} df​=λ2D2​

  • Fraunhofer zone, D: Antenna aperture

Three Propagation Mechanisms

  • Reflection + Diffraction + Scattering

Reflection

  • Object size >> λ
  • Ground, building surfaces

Diffraction

  • Sharp edges
  • Huygens principle→Wave bending

Scattering

  • Object size << λ
  • Rough surfaces→Many weak reflections

Log-distance Model

PL(d)=PL(d0)+10nlg⁡dd0PL(d) = PL(d_0) + 10n\lg\frac{d}{d_0} PL(d)=PL(d0​)+10nlgd0​d​

Path Loss Exponent n

  • Free space: n=2
  • Urban cellular: 2<n<5
  • Indoor NLOS: 4~6

MMSE Estimation of n

  • min Σ[PL_meas - PL(d₀) - 10n·lg(d/d₀)]²

Standard Deviation

σ=1N∑(PLmeas−PLpred)2\sigma = \sqrt{\frac{1}{N}\sum(PL_{meas}-PL_{pred})^2} σ=N1​∑(PLmeas​−PLpred​)2​

Log-normal Shadowing

  • PL(d) = PL̄(d) + Xσ
  • Xσ ~ N(0,σ²)

Q Function

  • P(Pr>γ) = Q[(γ-P̄r)/σ]
  • Q(x)=∫_x^∞ exp(-t²/2)/√(2π) dt

Coverage Probability

  • Boundary coverage U(γ)=Q[(γ-P̄r)/σ]
  • Area coverage > Boundary coverage
  • Example: Boundary 67.4%→Area 88%

Link Budget

  • Max PL = EIRP - Sensitivity - Shadowing Margin

4️⃣ Small-Scale Fading

Multipath Channel Model

  • y(t) = Σ aᵢ·x(t-τᵢ)

Doppler Shift

fd=vλcos⁡θ=vfcccos⁡θf_d = \frac{v}{\lambda}\cos\theta = \frac{v f_c}{c}\cos\theta fd​=λv​cosθ=cvfc​​cosθ

Time-varying Impulse Response

  • h(t,τ) = Σ aᵢ(t)exp[jφᵢ(t)]δ(τ-τᵢ(t))

Example Problem

  • fc=1850MHz,v=60mph=26.82m/s
  • fd,max=26.82/0.162≈165.6Hz

PDP Parameters

  • Power Delay Profile

Mean Excess Delay

τˉ=∑P(τk)τk∑P(τk)\bar{\tau} = \frac{\sum P(\tau_k)\tau_k}{\sum P(\tau_k)} τˉ=∑P(τk​)∑P(τk​)τk​​

RMS Delay Spread

στ=τ2‾−τˉ2\sigma_\tau = \sqrt{\overline{\tau^2} - \bar{\tau}^2} στ​=τ2−τˉ2​

Second Moment

τ2‾=∑P(τk)τk2∑P(τk)\overline{\tau^2} = \frac{\sum P(\tau_k)\tau_k^2}{\sum P(\tau_k)} τ2=∑P(τk​)∑P(τk​)τk2​​

Coherence Bandwidth/Time

  • Bc ≈ 1/(5στ) (Rhh=0.5)
  • Tc ≈ 0.423/fm

Maximum Doppler

  • fm = v/λ = v·fc/c

Rhh=0.9

  • Bc ≈ 1/(50στ)

Alternative Formula

  • Tc ≈ 9/(16π·fm)

Fading Type Determination

  • Bs vs Bc, Ts vs Tc

Flat Fading

  • στ < Ts and Bs << Bc
  • ISI negligible

Frequency Selective

  • στ > Ts and Bs >> Bc
  • Equalizer needed, severe ISI

Slow Fading

  • Tc >> Ts
  • Channel changes slower than symbols

Fast Fading

  • Tc << Ts
  • Time-selective fading

Equalization Condition

  • στ/Ts < 0.1 No equalizer needed
  • Example: στ=0.5μs→Ts>5μs→Rs<200ksps

Fading Distributions

  • Rayleigh vs Rician

Rayleigh

p(r)=rσ2e−r2/2σ2p(r) = \frac{r}{\sigma^2}e^{-r^2/2\sigma^2} p(r)=σ2r​e−r2/2σ2

  • No LOS, NLOS

Rician

p(r)=rσ2e−(r2+A2)/2σ2I0(rAσ2)p(r) = \frac{r}{\sigma^2}e^{-(r^2+A^2)/2\sigma^2}I_0(\frac{rA}{\sigma^2}) p(r)=σ2r​e−(r2+A2)/2σ2I0​(σ2rA​)

K-factor

  • K = A²/(2σ²)
  • K→0: Rayleigh
  • K→∞: No fading

Error Performance

  • AWGN: Pe∝exp(-SNR)
  • Rayleigh: Pe∝1/SNR

Anti-fading Techniques

  • Equalization + Diversity + Coding

Equalization

  • Compensate ISI
  • Adaptive inverse filtering
  • GSM application

Diversity Combining

  • Selection combining: Choose max SNR
  • EGC: Equal weights
  • MRC: Optimal weighting, γ=Σγᵢ

Diversity Types

  • Time/Frequency/Space
  • Transmit/Receive/Polarization

Rake Receiver

  • Separate multipaths(Δτ>Tc)
  • Coherent combining
  • CDMA uses PN code low autocorrelation

5️⃣ Multiple Access

Duplex Methods

  • FDD vs TDD

FDD

  • Uplink/downlink different frequencies
  • Simultaneous transmission
  • Requires duplexer

TDD

  • Uplink/downlink same frequency
  • Time division transmission
  • No duplexer needed

Note

  • TDD ≠ TDMA!
  • TDD: Duplex technique
  • TDMA: Multiple access technique

Four Major Multiple Access

  • FDMA / TDMA / CDMA / OFDMA

FDMA

  • Frequency division, each user exclusive frequency band
  • AMPS application
  • Requires RF filtering, intermodulation exists

TDMA

  • Time division, each user exclusive time slot
  • GSM application
  • Burst transmission, requires equalization

CDMA

  • Code division, same frequency simultaneous
  • IS-95/WCDMA
  • Soft capacity, soft handover

OFDMA

  • Orthogonal subcarriers
  • LTE-A/5G
  • Resistant to frequency selective fading, high spectral efficiency

Spread Spectrum Techniques

  • Gp = Bs/Bb = Rc/Rb = Tb/Tc

DSSS

  • Direct sequence: s(t)=d(t)·c(t)
  • Despreading: Signal compressed, interference spread

FHSS

  • Frequency hopping: Carrier frequency pseudo-random hopping
  • Anti-jamming: Only affects individual frequency points

Processing Gain

  • Gp = Spread bandwidth/Information bandwidth
  • Anti-jamming capability ∝ Gp

CDMA Key Technologies

  • Power Control + Soft Handover + Rake

Near-far Effect

  • Near-end strong signal suppresses far-end
  • Solution: Power control

Power Control

  • Open loop: MS estimates path loss
  • Closed loop: BS sends TPC commands

Soft Capacity

  • Users↑→Noise floor↑
  • No hard blocking

Self-interference

  • Codes not completely orthogonal
  • Multi-user detection/Power control

Random Access

  • Aloha / CSMA

Pure Aloha

  • Send anytime
  • Throughput~18.4%
  • Collision window 2T

Slotted Aloha

  • Send at slot start
  • Throughput~36.8%
  • Collision window T

CSMA/CA

  • Listen before talk
  • RTS/CTS handshake
  • 802.11 WiFi

CSMA/CD

  • Collision detection
  • 802.3 Ethernet
