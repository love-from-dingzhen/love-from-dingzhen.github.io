分析力学辛几何

我的群友为什么什么都学过

上一节讨论了向量空间上的辛形式。辛形式 Ω\OmegaΩ 给相空间 ZZZ 装上了一种“面积结构”。
这一节要回答一个更接近力学的问题：

什么样的变量变换，才不会破坏哈密顿力学的结构？\boxed{ \text{什么样的变量变换，才不会破坏哈密顿力学的结构？} } 什么样的变量变换，才不会破坏哈密顿力学的结构？​

在普通几何里，保持长度和角度的变换叫正交变换。
在辛几何里，保持辛形式 Ω\OmegaΩ 的变换叫辛映射。
而在分析力学里，它就是我们熟悉的正则变换。

不是所有变量替换都是力学上等价的

在哈密顿力学中，相空间点写成

z=(q,p).z=(q,p). z=(q,p).

如果换一组变量

z~=φ(z),\widetilde z=\varphi(z), z=φ(z),

也就是

(q~,p~)=φ(q,p),(\widetilde q,\widetilde p)=\varphi(q,p), (q​,p​)=φ(q,p),

表面上看只是重新命名了相空间坐标。

但是，并不是所有换变量都能保持哈密顿方程的形式。

哈密顿力学不是普通的微分方程系统。它依赖相空间上的辛结构 Ω\OmegaΩ。
因此，一个变量变换要想在力学上是“好的”，它不能只是一一对应，还必须保持 Ω\OmegaΩ。

所以真正的问题是：

变量变换 φ 是否保持相空间的辛面积结构？\boxed{ \text{变量变换 } \varphi \text{ 是否保持相空间的辛面积结构？} } 变量变换 φ 是否保持相空间的辛面积结构？​

如果保持，它就是正则变换；如果不保持，它就可能把哈密顿系统变成一个不再具有标准哈密顿形式的系统。

先引入几个几何记号：推前与拉回

为了精确表达“一个变换是否保持辛结构”，我们需要先说明：当一个映射把旧相空间送到新相空间时，向量、函数和辛形式应该怎样跟着变化。

设有一个光滑映射

f:Z→Y.f:Z\to Y. f:Z→Y.

这里 ZZZ 可以理解为旧相空间，YYY 可以理解为新相空间。
如果 z∈Zz\in Zz∈Z，那么它被送到

f(z)∈Y.f(z)\in Y. f(z)∈Y.

这只是点的变化。
但力学里更重要的是：点附近的方向、函数、辛形式如何变化。

导数映射 Df(z)Df(z)Df(z)：把旧方向推到新方向

在点 zzz 处，若有一个相空间方向

v∈Z,v\in Z, v∈Z,

那么经过映射 fff 后，它会变成 YYY 中的一个方向：

Df(z)⋅v∈Y.Df(z)\cdot v\in Y. Df(z)⋅v∈Y.

这里 Df(z)Df(z)Df(z) 是 fff 在点 zzz 处的导数，也可以理解成 fff 对无穷小方向的作用。

所以：

Df(z):Z→Y\boxed{ Df(z):Z\to Y } Df(z):Z→Y​

表示把旧空间中的无穷小方向推到新空间中。

直观地说，如果 vvv 是旧坐标中一个很小的位移方向，那么 Df(z)⋅vDf(z)\cdot vDf(z)⋅v 就是这个方向在新坐标中对应的位移方向。

函数的拉回 f∗f^*f∗

如果 YYY 上有一个函数

G:Y→R,G:Y\to\mathbb{R}, G:Y→R,

那么通过 fff，可以把它变成 ZZZ 上的函数：

f∗G=G∘f.f^*G=G\circ f. f∗G=G∘f.

也就是说，

(f∗G)(z)=G(f(z)).(f^*G)(z)=G(f(z)). (f∗G)(z)=G(f(z)).

这叫函数的拉回。

为什么叫“拉回”？
因为 GGG 本来是定义在新空间 YYY 上的，现在通过 fff 被拉回到了旧空间 ZZZ 上。

所以：

f∗G=G∘f\boxed{ f^*G=G\circ f } f∗G=G∘f​

表示把新空间上的函数拉回旧空间。

向量场的推前 f∗Xf_*Xf∗​X

如果 XXX 是 ZZZ 上的一个向量场，那么它在每一点 zzz 给出一个方向

X(z)∈Z.X(z)\in Z. X(z)∈Z.

经过映射 fff 后，这个方向被 Df(z)Df(z)Df(z) 推到 YYY 中：

Df(z)⋅X(z).Df(z)\cdot X(z). Df(z)⋅X(z).

于是可以定义推前向量场 f∗Xf_*Xf∗​X：

(f∗X)(f(z))=Df(z)⋅X(z).(f_*X)(f(z))=Df(z)\cdot X(z). (f∗​X)(f(z))=Df(z)⋅X(z).

也就是说：

f∗X 表示把旧空间中的向量场推到新空间。\boxed{ f_*X\text{ 表示把旧空间中的向量场推到新空间。} } f∗​X 表示把旧空间中的向量场推到新空间。​

如果把向量场看成“流动方向”，那么 f∗Xf_*Xf∗​X 就是在新变量中看到的流动方向。

辛形式的拉回 f∗Ξf^*\Xif∗Ξ

现在设 YYY 上有一个辛形式

Ξ.\Xi. Ξ.

它本来吃 YYY 中的两个向量。
但是我们也可以通过 fff 把它拉回到 ZZZ 上。

定义为

(f∗Ξ)z(v,w)=Ξf(z)(Df(z)⋅v,Df(z)⋅w).(f^*\Xi)_z(v,w) = \Xi_{f(z)}(Df(z)\cdot v,Df(z)\cdot w). (f∗Ξ)z​(v,w)=Ξf(z)​(Df(z)⋅v,Df(z)⋅w).

这句话的意思是：

先把 ZZZ 中的两个方向 v,wv,wv,w 通过 Df(z)Df(z)Df(z) 推到 YYY 中，然后用 YYY 上的辛形式 Ξ\XiΞ 计算它们的辛配对。

所以 f∗Ξf^*\Xif∗Ξ 是 ZZZ 上的一个双线性形式。

它的作用是：

把新空间上的辛形式拉回旧空间，然后和旧空间原来的辛形式比较。\boxed{ \text{把新空间上的辛形式拉回旧空间，然后和旧空间原来的辛形式比较。} } 把新空间上的辛形式拉回旧空间，然后和旧空间原来的辛形式比较。​

这正是判断一个变换是否保持辛结构的关键。

用拉回来定义辛映射和正则变换

有了上面的记号之后，就可以很简洁地说清楚什么是辛映射。

设 (Z,Ω)(Z,\Omega)(Z,Ω) 和 (Y,Ξ)(Y,\Xi)(Y,Ξ) 是两个辛向量空间，映射

f:Z→Yf:Z\to Y f:Z→Y

称为辛映射，如果它满足

f∗Ξ=Ω.f^*\Xi=\Omega. f∗Ξ=Ω.

展开来说，就是对任意 z∈Zz\in Zz∈Z 和任意两个方向 v,w∈Zv,w\in Zv,w∈Z，都有

Ξ(Df(z)⋅v,Df(z)⋅w)=Ω(v,w).\Xi(Df(z)\cdot v,Df(z)\cdot w)=\Omega(v,w). Ξ(Df(z)⋅v,Df(z)⋅w)=Ω(v,w).

这表示：
旧空间中的两个方向 v,wv,wv,w 先经过 Df(z)Df(z)Df(z) 变成新空间中的两个方向，然后用新空间的辛形式 Ξ\XiΞ 计算辛面积，结果必须和旧空间中用 Ω\OmegaΩ 计算的辛面积完全相同。

所以辛映射的几何意义是：

辛映射保持相空间中的辛面积结构。\boxed{ \text{辛映射保持相空间中的辛面积结构。} } 辛映射保持相空间中的辛面积结构。​

它不一定保持普通长度，也不一定保持普通角度。
它真正保持的是由 qqq 和 ppp 交叉配对形成的辛结构。

正则变换是相空间到自身的辛映射

在哈密顿力学中，我们通常考虑同一个相空间上的变量替换：

φ:Z→Z.\varphi:Z\to Z. φ:Z→Z.

如果 ZZZ 上的辛形式是 Ω\OmegaΩ，那么 φ\varphiφ 是正则变换，当且仅当

φ∗Ω=Ω.\varphi^*\Omega=\Omega. φ∗Ω=Ω.

展开就是

Ω(Dφ(z)⋅v,Dφ(z)⋅w)=Ω(v,w).\Omega(D\varphi(z)\cdot v,D\varphi(z)\cdot w)=\Omega(v,w). Ω(Dφ(z)⋅v,Dφ(z)⋅w)=Ω(v,w).

这说明正则变换不是普通的坐标替换，而是保持辛形式的坐标替换。

换句话说：

正则变换就是保持辛结构的变量变换。\boxed{ \text{正则变换就是保持辛结构的变量变换。} } 正则变换就是保持辛结构的变量变换。​

从力学角度看，它保持的是 qqq 与 ppp 作为共轭变量的关系。
所以换到新变量以后，系统仍然可以写成哈密顿方程的形式。

正则变换为什么重要

正则变换的定义可以得出一个很漂亮的结论:哈密顿方程的结构不变
不妨在这种变换下，哈密顿变成了

H~=H~φ−1H~(z~)=H(z).\widetilde H= \widetilde H\varphi^{-1}\quad \quad \widetilde H(\widetilde z)=H(z). H=Hφ−1H(z)=H(z).

Ω(XH,v)=dH(v)\Omega(X_H,v)=dH(v) Ω(XH​,v)=dH(v)

Ω(Dφ(z)⋅XH,Dφ(z)⋅v)=Ω(XH,v).\Omega(D\varphi(z)\cdot X_H,D\varphi(z)\cdot v)=\Omega(X_H,v). Ω(Dφ(z)⋅XH​,Dφ(z)⋅v)=Ω(XH​,v).

dH(v)=dH~(v~)=dH~(Dφ(z)⋅v)dH(v)= d\widetilde H(\widetilde v) = d\widetilde H\bigl(D\varphi(z)\cdot v\bigr) dH(v)=dH(v)=dH(Dφ(z)⋅v)

φ∗XH=XH~.\boxed{ \varphi_*X_H=X_{\widetilde H}. } φ∗​XH​=XH​.​

这一步说明：正则变换会把原来的哈密顿向量场推成新哈密顿量对应的哈密顿向量场。

所以，在正则变换下

z˙=XH(z)z~˙=XH~(z~).\dot z=X_H(z)\quad\quad\quad \dot{\widetilde z}=X_{\widetilde H}(\widetilde z). z˙=XH​(z)z˙=XH​(z).

线性情形：矩阵条件 ATJA=JA^TJA=JATJA=J

如果变换是线性的(当然，在每个点的局域内都近似线性)，记为

z~=Az,\widetilde z=Az, z=Az,

那么导数就是矩阵 AAA 本身。

在标准相空间中，辛形式写成

Ω(v,w)=vTJw,\Omega(v,w)=v^T Jw, Ω(v,w)=vTJw,

其中

J=(0I−I0).J= \begin{pmatrix} 0&I\\ -I&0 \end{pmatrix}. J=(0−I​I0​).

辛映射条件是

Ω(Av,Aw)=Ω(v,w),∀v,w∈Z.\Omega(Av,Aw)=\Omega(v,w), \qquad \forall v,w\in Z. Ω(Av,Aw)=Ω(v,w),∀v,w∈Z.

(这里的A着几何形式中的雅可比矩阵)
代入矩阵形式：

(Av)TJ(Aw)=vTJw.(Av)^T J(Aw)=v^T Jw. (Av)TJ(Aw)=vTJw.

于是得到

vTATJAw=vTJw.v^T A^T J A w=v^T Jw. vTATJAw=vTJw.

ATJA=J.\boxed{ A^TJA=J. } ATJA=J.​

这就是线性辛变换的矩阵条件。

它的几何意义是：

A 可以拉伸、扭曲、旋转相空间，但不能改变辛面积结构。\boxed{ A\text{ 可以拉伸、扭曲、旋转相空间，但不能改变辛面积结构。} } A 可以拉伸、扭曲、旋转相空间，但不能改变辛面积结构。​

与普通面积保持变换的区别

在一维相空间中，Z=R2Z=\mathbb{R}^2Z=R2，辛形式就是

Ω=dq∧dp.\Omega=dq\wedge dp. Ω=dq∧dp.

这时保持 Ω\OmegaΩ 基本上就是保持 qqq-ppp 平面中的有向面积。

但在高维相空间中，情况更强。
若系统有 nnn 个自由度，相空间维数是 2n2n2n。辛形式为

Ω=∑i=1ndqi∧dpi.\Omega=\sum_{i=1}^n dq^i\wedge dp_i. Ω=i=1∑n​dqi∧dpi​.

辛映射不仅要保持总体积，还要保持每一对共轭方向之间的结构。

因此，辛映射比一般的体积保持映射更严格。

可以这样说：

辛映射一定保持相体积，但保持相体积的映射不一定是辛映射。\boxed{ \text{辛映射一定保持相体积，但保持相体积的映射不一定是辛映射。} } 辛映射一定保持相体积，但保持相体积的映射不一定是辛映射。​

因为辛映射保存的不只是“大小”，而是 qqq 与 ppp 的配对方式。

ATJA=J.A^TJA=J. ATJA=J.

从物理上说，这表示：
位置和动量的混合是必然的，因为任意力学系统随时间的演化都是一串无穷小的辛变换

一个简单例子：交换 qqq 和 ppp

在一维相空间中，考虑变换

(q,p)↦(p,−q).(q,p)\mapsto (p,-q). (q,p)↦(p,−q).

它的矩阵是

A=(01−10)=J.A= \begin{pmatrix} 0&1\\ -1&0 \end{pmatrix} =J. A=(0−1​10​)=J.

这个变换把位置和动量交换，同时加上一个负号。

为什么需要负号？
因为如果只是

(q,p)↦(p,q),(q,p)\mapsto(p,q), (q,p)↦(p,q),

那么

dq∧dpdq\wedge dp dq∧dp

会变成

dp∧dq=−dq∧dp.dp\wedge dq=-dq\wedge dp. dp∧dq=−dq∧dp.

辛形式的符号被翻转了，结构没有保持。

而

(q,p)↦(p,−q)(q,p)\mapsto(p,-q) (q,p)↦(p,−q)

会保持

dq∧dp.dq\wedge dp. dq∧dp.

所以这个负号不是随便的，它正是为了保持辛面积的方向。

这说明正则变换关心的不只是面积大小，还关心有向面积的符号和 qqq-ppp 的配对方向。

另一个例子：动量平移 p↦p+df(q)p\mapsto p+df(q)p↦p+df(q)

设 f(q)f(q)f(q) 是一个只依赖位置的函数。考虑变换

(q,p)↦(q,p+df(q)).(q,p)\mapsto(q,p+df(q)). (q,p)↦(q,p+df(q)).

这类变换在力学中很常见。它改变动量，却不改变位置。

几何上，它可以看成在每个位置点 qqq 上，把动量空间平移一个协向量 df(q)df(q)df(q)。

为什么它仍然可能是正则变换？
因为 dfdfdf 是一个精确 111-形式，而

d(df)=0.d(df)=0. d(df)=0.

它不会改变基本辛形式

dq∧dp.dq\wedge dp. dq∧dp.

物理上，这和拉格朗日量加上一个全导数有关。
加全导数不会改变运动方程，但会改变正则动量的表达。

所以这类动量平移说明：

正则变换允许改变动量的表达，但不允许改变相空间的辛结构。\boxed{ \text{正则变换允许改变动量的表达，但不允许改变相空间的辛结构。} } 正则变换允许改变动量的表达，但不允许改变相空间的辛结构。​
