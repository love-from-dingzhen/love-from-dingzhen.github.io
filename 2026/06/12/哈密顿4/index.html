去复习期末了

前面已经知道，辛形式 Ω\OmegaΩ 给相空间 ZZZ 装上了一种特殊的“面积结构”。
正则变换的本质，就是保持这种辛面积结构不变。

有了辛形式 Ω 和哈密顿量 H，怎样得到运动方程？\boxed{ \text{有了辛形式 }\Omega\text{ 和哈密顿量 }H\text{，怎样得到运动方程？} } 有了辛形式 Ω 和哈密顿量 H，怎样得到运动方程？​

1. 为什么要写成一般形式

在正则坐标中，哈密顿方程写成

q˙i=∂H∂pi,p˙i=−∂H∂qi.\dot q^i=\frac{\partial H}{\partial p_i}, \qquad \dot p_i=-\frac{\partial H}{\partial q^i}. q˙​i=∂pi​∂H​,p˙​i​=−∂qi∂H​.

这个形式很熟悉，但它依赖于具体坐标 (q,p)(q,p)(q,p)。

辛几何更关心的是：
如果没有提前指定某一组正则坐标，甚至相空间不是简单的 R2n\mathbb{R}^{2n}R2n，我们还能不能写出哈密顿方程？

答案是可以的。
真正本质的不是坐标公式，而是下面这条关系：

Ω(XH,v)=dH(v).\Omega(X_H,v)=dH(v). Ω(XH​,v)=dH(v).

这里：

  • HHH 是哈密顿量；
  • dHdHdH 是 HHH 的微分；
  • XHX_HXH​ 是哈密顿向量场；
  • Ω\OmegaΩ 是相空间上的辛形式；
  • vvv 是任意相空间方向。

这句话的意思是：

哈密顿向量场 XH 是那个通过辛形式与 dH 对应起来的方向。\boxed{ \text{哈密顿向量场 }X_H\text{ 是那个通过辛形式与 }dH\text{ 对应起来的方向。} } 哈密顿向量场 XH​ 是那个通过辛形式与 dH 对应起来的方向。​

也就是说，运动方向不是随便选的，而是由 HHH 和 Ω\OmegaΩ 共同决定的。

2. dHdHdH 不是运动方向

哈密顿量

可以理解为相空间上的能量函数。

在某一点 z∈Zz\in Zz∈Z，它的微分

表示能量在各个方向上的变化率。
如果给定一个方向 vvv，那么

但是要注意：

6. 为什么运动不是沿着能量梯度走

如果在普通欧氏空间里，给一个能量函数 HHH，最直接的方向是梯度方向

梯度方向表示 HHH 增加最快的方向。

但哈密顿系统不是梯度流。
它的运动方向是

XH=J∇H.X_H=J\nabla H. XH​=J∇H.

也就是说，辛结构 JJJ 把 ∇H\nabla H∇H 转成另一个方向。

一维时，

J=(01 −10).J= \begin{pmatrix} 0&1\ -1&0 \end{pmatrix}. J=(0​1 −1​0​).

所以直观上：

  • ∇H\nabla H∇H 指向能量增加最快的方向；
  • XH=J∇HX_H=J\nabla HXH​=J∇H 指向能量等值曲线的切方向。

因此哈密顿系统不是沿着能量上升或下降，而是在能量面上流动。

这就是为什么保守系统的相轨道往往沿着

7. 一般哈密顿方程的物理意义

一般哈密顿方程

首先，dH(v)dH(v)dH(v) 表示：如果系统沿着方向 vvv 发生无穷小变化，能量会如何改变。

其次，Ω(XH,v)\Omega(X_H,v)Ω(XH​,v) 表示：真实运动方向 XHX_HXH​ 和测试方向 vvv 之间的辛面积配对。

也就是说，真实运动方向 XHX_HXH​ 是被这样规定出来的：

它和任意方向的辛面积配对，必须精确反映能量微分。\boxed{ \text{它和任意方向的辛面积配对，必须精确反映能量微分。} } 它和任意方向的辛面积配对，必须精确反映能量微分。​

所以哈密顿力学的结构不是：

这就是辛几何与哈密顿力学真正联系起来的地方。

9. 能量守恒为什么自动成立

设 c(t)c(t)c(t) 是哈密顿方程的一条解曲线：

10. 一般形式为什么比坐标形式更重要

正则坐标下的哈密顿方程很好用：

q˙i=∂H∂pi,p˙i=−∂H∂qi.\dot q^i=\frac{\partial H}{\partial p_i}, \qquad \dot p_i=-\frac{\partial H}{\partial q^i}. q˙​i=∂pi​∂H​,p˙​i​=−∂qi∂H​.

但它掩盖了背后的几何结构。

一般形式

iXHΩ=dHi_{X_H}\Omega=dH iXH​​Ω=dH

更重要，因为它不依赖具体坐标。

它告诉我们，只要给定：

就可以定义哈密顿运动。

这里：

  • ZZZ 是相空间；
  • Ω\OmegaΩ 是辛形式；
  • HHH 是哈密顿量。

这三者共同决定系统：

(Z,Ω,H)⟹XH⟹运动轨道.\boxed{ (Z,\Omega,H) \quad\Longrightarrow\quad X_H \quad\Longrightarrow\quad \text{运动轨道}. } (Z,Ω,H)⟹XH​⟹运动轨道.​

所以哈密顿力学的本质不是某一组坐标公式，而是辛空间上的流。

12. 小结

经典写法是

q˙i=∂H∂pi,p˙i=−∂H∂qi.\dot q^i=\frac{\partial H}{\partial p_i}, \qquad \dot p_i=-\frac{\partial H}{\partial q^i}. q˙​i=∂pi​∂H​,p˙​i​=−∂qi∂H​.
