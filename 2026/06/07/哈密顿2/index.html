分析力学辛几何

学这种东西有什么用

这一节只讨论一个问题：

什么样的双线性形式可以叫做辛形式？\boxed{ \text{什么样的双线性形式可以叫做辛形式？} } 什么样的双线性形式可以叫做辛形式？​

在欧氏几何里，空间上最重要的结构是内积。内积告诉我们长度、角度和正交性。
但在哈密顿力学的相空间里，更自然的结构不是内积，而是辛形式。

内积关心的是

⟨v,w⟩,\langle v,w\rangle, ⟨v,w⟩,

它偏向于描述“距离几何”。

辛形式关心的是

Ω(v,w),\Omega(v,w), Ω(v,w),

它更像是在描述两个方向张成的“有向面积”。

1. 辛形式不是内积

设 ZZZ 是一个实向量空间。先把 ZZZ 理解成一个相空间的线性模型。

一个双线性形式是指

Ω:Z×Z→R.\Omega:Z\times Z\to \mathbb{R}. Ω:Z×Z→R.

它吃两个向量 v,w∈Zv,w\in Zv,w∈Z，吐出一个实数 Ω(v,w)\Omega(v,w)Ω(v,w)。

如果是内积，我们通常有

⟨v,w⟩=⟨w,v⟩.\langle v,w\rangle=\langle w,v\rangle. ⟨v,w⟩=⟨w,v⟩.

也就是说，内积是对称的。

但辛形式要求相反：

Ω(v,w)=−Ω(w,v).\Omega(v,w)=-\Omega(w,v). Ω(v,w)=−Ω(w,v).

这叫斜对称性。

特别地，令 w=vw=vw=v，就得到

Ω(v,v)=0.\Omega(v,v)=0. Ω(v,v)=0.

这说明辛形式不能像内积那样测量一个向量自己的长度。
它真正测量的是两个不同方向之间的交叉关系。

在二维平面里，最典型的例子是

v=(a,b),w=(c,d).v=(a,b),\qquad w=(c,d). v=(a,b),w=(c,d).

定义

Ω(v,w)=ad−bc.\Omega(v,w)=ad-bc. Ω(v,w)=ad−bc.

这正是二维平面中两个向量张成的有向面积。

所以辛形式的第一层几何意义是：

Ω 不是测长度的，而是测有向面积的。\boxed{ \Omega\text{ 不是测长度的，而是测有向面积的。} } Ω 不是测长度的，而是测有向面积的。​

2. 非退化性：不能有“看不见的方向”

辛形式除了斜对称，还必须非退化。

非退化的意思是：如果某个向量 z1∈Zz_1\in Zz1​∈Z 和所有向量 z2∈Zz_2\in Zz2​∈Z 的辛配对都为零，

Ω(z1,z2)=0,∀z2∈Z,\Omega(z_1,z_2)=0,\qquad \forall z_2\in Z, Ω(z1​,z2​)=0,∀z2​∈Z,

那么只能有

z1=0.z_1=0. z1​=0.

这句话的直观意思是：

空间中不能存在一个非零方向，却对 Ω 完全不可见。\boxed{ \text{空间中不能存在一个非零方向，却对 }\Omega\text{ 完全不可见。} } 空间中不能存在一个非零方向，却对 Ω 完全不可见。​

如果存在某个非零方向 z1z_1z1​，使得它和所有方向的辛配对都为零，那么 Ω\OmegaΩ 就无法识别这个方向。
这样的形式是退化的，不能作为真正的辛结构。

因此，辛形式要求：

每一个非零方向，都必须能被某个方向通过 Ω 检测出来。\boxed{ \text{每一个非零方向，都必须能被某个方向通过 }\Omega\text{ 检测出来。} } 每一个非零方向，都必须能被某个方向通过 Ω 检测出来。​

3. 非退化性可以用 Ω♭\Omega^\flatΩ♭ 表达

前面说过，Ω\OmegaΩ 非退化的意思是：

Ω(z1,z2)=0,∀z2∈Z⟹z1=0.\Omega(z_1,z_2)=0,\quad \forall z_2\in Z \quad\Longrightarrow\quad z_1=0. Ω(z1​,z2​)=0,∀z2​∈Z⟹z1​=0.

用 Ω♭\Omega^\flatΩ♭ 来说，就是：

Ω♭(z1)=0⟹z1=0.\Omega^\flat(z_1)=0 \quad\Longrightarrow\quad z_1=0. Ω♭(z1​)=0⟹z1​=0.

也就是说，Ω♭\Omega^\flatΩ♭ 至少应该是单射。

如果 ZZZ 是有限维空间，那么单射自动意味着满射。
所以在有限维情形中，Ω♭\Omega^\flatΩ♭ 是同构：

Ω♭:Z→Z∗.\Omega^\flat:Z\to Z^*. Ω♭:Z→Z∗.

这说明辛形式提供了一种在 ZZZ 和 Z∗Z^*Z∗ 之间相互转换的方式。

在无限维空间中，情况更细。
单射和满射不一定等价，所以会区分弱非退化和强非退化。

如果 Ω♭\Omega^\flatΩ♭ 是单射，称 Ω\OmegaΩ 弱非退化。
如果 Ω♭\Omega^\flatΩ♭ 是同构，称 Ω\OmegaΩ 强非退化。

这一区分主要在无限维力学系统中重要。对普通有限维分析力学来说，可以先把非退化理解成 Ω♭\Omega^\flatΩ♭ 可逆。

4. 辛向量空间

现在可以定义辛向量空间。

如果 ZZZ 上有一个双线性形式 Ω\OmegaΩ，并且满足：

Ω(v,w)=−Ω(w,v),\Omega(v,w)=-\Omega(w,v), Ω(v,w)=−Ω(w,v),

以及非退化条件，那么称

(Z,Ω)(Z,\Omega) (Z,Ω)

为一个辛向量空间。

也就是说，辛向量空间不是单纯的向量空间，而是带有一个特殊面积结构的向量空间。

可以对比：

欧氏向量空间=(V,⟨⋅,⋅⟩),\text{欧氏向量空间}=(V,\langle\cdot,\cdot\rangle), 欧氏向量空间=(V,⟨⋅,⋅⟩),

其中 ⟨⋅,⋅⟩\langle\cdot,\cdot\rangle⟨⋅,⋅⟩ 是内积。

而辛向量空间是

辛向量空间=(Z,Ω),\text{辛向量空间}=(Z,\Omega), 辛向量空间=(Z,Ω),

其中 Ω\OmegaΩ 是辛形式。

一个偏向长度和角度，一个偏向有向面积和配对结构。

5. 为什么有限维辛空间一定是偶数维

设 ZZZ 是有限维向量空间。取一组基 eI{e_I}eI​，定义

ΩIJ=Ω(eI,eJ).\Omega_{IJ}=\Omega(e_I,e_J). ΩIJ​=Ω(eI​,eJ​).

如果

z=zIeI,w=wJeJ,z=z^I e_I,\qquad w=w^J e_J, z=zIeI​,w=wJeJ​,

那么

Ω(z,w)=zIΩIJwJ.\Omega(z,w)=z^I\Omega_{IJ}w^J. Ω(z,w)=zIΩIJ​wJ.

由于 Ω\OmegaΩ 是斜对称的，所以

ΩIJ=−ΩJI.\Omega_{IJ}=-\Omega_{JI}. ΩIJ​=−ΩJI​.

因此矩阵 [ΩIJ][\Omega_{IJ}][ΩIJ​] 是斜对称矩阵。

非退化性意味着

det⁡[ΩIJ]≠0.\det[\Omega_{IJ}]\neq 0. det[ΩIJ​]=0.

但是有限维斜对称矩阵要想非退化，维数必须是偶数。
所以有限维辛向量空间必然是偶数维的。

这件事对力学很重要。
因为哈密顿相空间中的变量总是成对出现：

(q1,…,qn,p1,…,pn).(q^1,\dots,q^n,p_1,\dots,p_n). (q1,…,qn,p1​,…,pn​).

每个位置变量 qiq^iqi 都配有一个动量变量 pip_ipi​，所以相空间维数是

2n.2n. 2n.

从辛几何角度看，qqq 和 ppp 成对出现不是记号习惯，而是由非退化斜对称结构决定的。

标准例子：W×W∗W\times W^*W×W∗ 上的辛形式

最自然的辛向量空间来自一个向量空间 WWW 和它的对偶空间 W∗W^*W∗。

令

Z=W×W∗.Z=W\times W^*. Z=W×W∗.

一个元素可以写成

(w,α),(w,\alpha), (w,α),

其中

w∈W,α∈W∗.w\in W,\qquad \alpha\in W^*. w∈W,α∈W∗.

在 ZZZ 上定义

Ω((w1,α1),(w2,α2))=α2(w1)−α1(w2).\Omega((w_1,\alpha_1),(w_2,\alpha_2)) = \alpha_2(w_1)-\alpha_1(w_2). Ω((w1​,α1​),(w2​,α2​))=α2​(w1​)−α1​(w2​).

这就是标准正则辛形式。

如果把 WWW 理解成位置方向，把 W∗W^*W∗ 理解成动量方向，那么 W×W∗W\times W^*W×W∗ 就是相空间的线性版本。

标准矩阵形式

若 WWW 是 nnn 维空间，选取 WWW 的基

e1,…,en,{e_1,\dots,e_n}, e1​,…,en​,

以及 W∗W^*W∗ 的对偶基

e1,…,en.{e^1,\dots,e^n}. e1,…,en.

在 Z=W×W∗Z=W\times W^*Z=W×W∗ 中，一个向量可以写成

z=(q,p),z=(q,p), z=(q,p),

另一个向量写成

w=(Q,P).w=(Q,P). w=(Q,P).

这时标准辛形式为

Ω(z,w)=q⋅P−p⋅Q.\Omega(z,w)=q\cdot P-p\cdot Q. Ω(z,w)=q⋅P−p⋅Q.

矩阵写法是

Ω(z,w)=zTJw,\Omega(z,w)=z^T Jw, Ω(z,w)=zTJw,

其中

J=(0I−I0).J= \begin{pmatrix} 0&I\\ -I&0 \end{pmatrix}. J=(0−I​I0​).

所以 JJJ 不是凭空出现的符号，而是标准辛形式在正则坐标下的矩阵表示。

一维时，

z=(q,p),w=(Q,P),z=(q,p),\qquad w=(Q,P), z=(q,p),w=(Q,P),

于是

Ω(z,w)=qP−pQ.\Omega(z,w)=qP-pQ. Ω(z,w)=qP−pQ.

这就是二维相平面中的有向面积。

与内积空间的关系

如果 WWW 是一个实内积空间，内积记为

⟨⋅,⋅⟩,\langle\cdot,\cdot\rangle, ⟨⋅,⋅⟩,

那么可以用内积把 WWW 是自对偶。

于是 W×W∗W\times W^*W×W∗ 可以写成 W×WW\times WW×W。

在 Z=W×WZ=W\times WZ=W×W 上，可以定义

Ω((w1,w2),(z1,z2))=⟨z2,w1⟩−⟨z1,w2⟩.\Omega((w_1,w_2),(z_1,z_2)) = \langle z_2,w_1\rangle-\langle z_1,w_2\rangle. Ω((w1​,w2​),(z1​,z2​))=⟨z2​,w1​⟩−⟨z1​,w2​⟩.

这和 W×W∗W\times W^*W×W∗ 上的正则辛形式是同一个结构，只是借助内积把协向量写成了向量。

例如 W=R3W=\mathbb{R}^3W=R3 时，一个质点的相空间可以写成

R3×R3.\mathbb{R}^3\times\mathbb{R}^3. R3×R3.

第一个 R3\mathbb{R}^3R3 对应位置方向，第二个 R3\mathbb{R}^3R3 对应动量方向。

这时辛形式就是位置变化与动量变化之间的交叉配对。

复空间中的辛形式

复空间 Cn\mathbb{C}^nCn 也天然带有辛形式。

设

z=(z1,…,zn),w=(w1,…,wn),z=(z_1,\dots,z_n),\qquad w=(w_1,\dots,w_n), z=(z1​,…,zn​),w=(w1​,…,wn​),

标准厄米内积为

⟨z,w⟩=∑j=1nzjwj‾.\langle z,w\rangle = \sum_{j=1}^n z_j\overline{w_j}. ⟨z,w⟩=j=1∑n​zj​wj​​.

这个复内积的实部给出普通度量：

Re⁡⟨z,w⟩.\operatorname{Re}\langle z,w\rangle. Re⟨z,w⟩.

而虚部给出辛结构。通常可以定义

Ω(z,w)=−Im⁡⟨z,w⟩.\Omega(z,w) = -\operatorname{Im}\langle z,w\rangle. Ω(z,w)=−Im⟨z,w⟩.

因此，在复空间中，同一个厄米内积同时包含两种结构：

Re⁡⟨z,w⟩ 给出长度和角度，\boxed{ \operatorname{Re}\langle z,w\rangle \text{ 给出长度和角度，} } Re⟨z,w⟩ 给出长度和角度，​

−Im⁡⟨z,w⟩ 给出辛面积。\boxed{ -\operatorname{Im}\langle z,w\rangle \text{ 给出辛面积。} } −Im⟨z,w⟩ 给出辛面积。​

这说明复结构、度量结构和辛结构之间有天然联系。
